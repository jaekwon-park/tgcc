# Contributing to tgcc

현재 v0.1 MVP 설계 단계입니다. 아래 형태로 도움이 가능합니다.

## 단계별 환영하는 기여

### 지금 (설계 단계)

- 문서 리뷰: `docs/` 안의 결정 사항에 대한 의견·반박
- 누락된 시나리오 제보 (이슈)
- ACL 모델, 상태 머신, SQLite 스키마 검증

### v0.1 구현 시작 후

- 마일스톤 M1~M5 코드 PR
- 테스트 케이스 추가
- macOS/Linux 빌드 검증

### v0.2 이후

- Honcho 통합 개선
- TUI 대시보드 (bubbletea 등)
- 권한 승인 인라인 키보드 UI

## 토론 가이드

- 큰 설계 변경은 이슈로 먼저 제안
- 비용 모델(구독 한도 vs Agent SDK 크레딧) 가정을 깨는 변경은 반드시 사전 합의 필요
- 외부 의존성 추가는 최소화 (단일 바이너리 원칙)

## 코딩 스타일 (v0.1 작성 시작 시 확정)

- Go 1.22+
- 표준 라이브러리 우선
- `log/slog` 구조화 로깅
- 인터페이스는 사용처에 정의
- 패키지명은 단수형 (`session`, `bot`, `acl`)
